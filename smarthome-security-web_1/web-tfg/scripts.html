Scripts de Auditoría

Herramientas desarrolladas como parte del proyecto para automatizar la detección de vulnerabilidades en redes domésticas. Listas para usar en tu propia red.

⚠️

Aviso importante sobre el uso ético

Estos scripts están diseñados para auditar tu propia red. Ejecutarlos sobre redes que no son tuyas o sin consentimiento explícito del propietario es ilegal en la mayoría de países y puede acarrear consecuencias legales graves. Úsalos responsablemente.

Descarga los scripts

Dos versiones completas: una para sistemas Linux (Bash) y otra para Windows (PowerShell). Ambas realizan un análisis inicial de la seguridad de tu red.

🐧

Linux · Bash

smarthome-audit.sh · v1.0

Script de auditoría para sistemas Linux (probado en Kali, Ubuntu y Debian). Automatiza escaneos de red, análisis WiFi y detección de servicios expuestos.

  • Escaneo de la red con nmap
  • Análisis de configuración WiFi
  • Detección de puertos y servicios abiertos
  • Comprobación de contraseñas débiles
  • Informe automático en TXT y HTML
  • Modo verbose y silencioso
Tamaño
~12 KB
Licencia
MIT
Requisitos
bash, nmap
⬇️ Descargar (ZIP)

Uso rápido (después de descomprimir):

chmod +x smarthome-audit.sh && sudo ./smarthome-audit.sh

Windows · PowerShell

smarthome-audit.ps1 · v1.0

Versión para sistemas Windows 10/11. Aprovecha PowerShell para realizar un análisis equivalente sin necesidad de instalar herramientas externas.

  • Escaneo de la red local
  • Análisis de perfiles WiFi guardados
  • Detección de puertos con PowerShell nativo
  • Comprobación del Firewall de Windows
  • Informe HTML generado automáticamente
  • No requiere instalaciones adicionales
Tamaño
~10 KB
Licencia
MIT
Requisitos
PS 5.1+

Uso previsto:

Set-ExecutionPolicy RemoteSigned && .\smarthome-audit.ps1

¿Cómo funcionan?

Los scripts siguen una metodología estándar de auditoría en 4 fases

01
🔍

Reconocimiento

Detectan la red local, la puerta de enlace, el rango de IPs y los dispositivos conectados. Todo sin ser intrusivo.

→
02
📡

Análisis

Escanean puertos, identifican servicios, detectan configuraciones WiFi inseguras (WPS, WPA2 débil, SSID visibles).

→
03
⚖️

Evaluación

Comparan hallazgos con una base de conocimiento interna y clasifican los problemas por criticidad.

→
04
📄

Informe

Generan un informe con los resultados, vulnerabilidades encontradas y recomendaciones concretas para corregirlas.
